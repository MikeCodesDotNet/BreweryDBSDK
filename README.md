# [BreweryDBSDK]

BreweryDBSDK is an open source library for interacting with BreweryDB.com (http://www.brewerydb.com).


## Unique to this fork


* Searching for beers by name
* Searching for breweries by name
* MonoTouch support

* Implentation of the entire BreweryDB API
